# Fractal Explorer

**Fractal Explorer** is an interactive, browser-based Mandelbrot and Julia set visualizer. It allows real-time zooming, coloring, and exploration of complex fractals using various rendering and coloring techniques.

## 🌌 What Are Fractals?

Fractals are infinitely complex patterns that are self-similar across different scales. They are created by repeating a simple mathematical process over and over. The most famous example is the Mandelbrot Set, which is generated by iterating the function:

`z = z² + c`


Fractals reveal intricate and beautiful patterns that emerge from relatively simple mathematical rules. They are used in everything from nature modeling and computer graphics to signal processing and chaos theory.

## 🧭 How to Use the App

### 🖱️ Mouse Controls

- **Left click** on the canvas to **zoom in** centered on the mouse pointer.
- **Right click** to **zoom out** from the clicked point.
- Zooming is animated smoothly and retains the focus on the clicked location.

### 🎛️ Controls Panel

The sidebar contains multiple options to customize the fractal:

| Control         | Description |
|----------------|-------------|
| **Fractal Type** | Choose between `Mandelbrot` and `Julia`. |
| **Julia Real / Imag** | Set the complex constant **c** (only for Julia sets). |
| **Max Iterations** | Defines how many times the function is iterated for each pixel. Higher values show more detail but take longer to render. |
| **Coloring** | Choose the algorithm for how fractal values are mapped to colors. Options include `binary`, `escape`, `smooth`, `gradient`. |
| **Color Scheme** | Pick a color palette for rendering: `classic`, `fire`, `fern`, `ice`, `plasma`, `pastel`, `neon`, and more. |
| **Reset View** | Resets zoom and position to show the full fractal. |
| **Download Image** | Saves the current fractal view as a `.png` file. |

## 🎨 Coloring Algorithms

The app supports multiple algorithms for coloring fractals:

- **Binary**: Black & white based on escape or non-escape.
- **Escape Time**: Linear color based on how quickly points escape.
- **Smooth Coloring**: Uses log smoothing to avoid banding.
- **Gradient Mapping**: Maps normalized iteration counts to continuous color gradients for smooth visual flow.

Each coloring mode works with a variety of color palettes.

## 🌈 Color Schemes

The following color schemes are available:

- `Classic` – Sinusoidal RGB cycle
- `Fire` – Hot reds and oranges
- `Fern` – Natural greens and soft earth tones
- `Ice` – Cool blues and teals
- `Neon` – High-contrast bright colors
- `Pastel` – Soft and pale colors
- `Plasma` – Retro purples and electric pinks
- `Sunset` – Orange, magenta, violet blends
- `Ocean` – Deep blue gradients with aqua highlights
